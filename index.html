The Responsive Calculator

turning pixels into percentages

Just a simple calculator to help turn your PSD pixel perfection into the start of your responsive website.

* {

-webkit-box-sizing: border-box; /* Safari/Chrome, other WebKit */

-moz-box-sizing: border-box; /* Firefox, other Gecko */

box-sizing: border-box; /* Opera/IE 8+ */

}

Created with love, tears and help by @sturobson

This is just an initial release, I'd love for this to do so much more. Can you help?
